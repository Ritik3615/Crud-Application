# Full Stack CRUD Application  
### React (Vite) + Spring Boot + PostgreSQL (Neon DB)

This is a full-stack CRUD project built using **React (Vite)** for the frontend and **Spring Boot** for the backend.  
The application supports creating, reading, updating, and deleting user records with proper validations, alerts, and clean API communication.

---

## üìå Features

### **Frontend**
- Create User (with SweetAlert success messages)
- Read User (table view with loading spinner)
- Update User (prefilled form + confirmation popup)
- Delete User (confirmation popup + soft delete)
- Background video support (Update page example)
- Clean UI with Tailwind CSS

### **Backend**
- REST API Endpoints for CRUD
- Global exception handling
- Duplicate email validation
- Soft delete support
- Auto timestamping (`createdAt`, `updatedAt`)
- CORS enabled for Vite frontend
- Full security disabled for development
